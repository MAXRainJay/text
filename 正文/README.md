# LaTeX 论文工程说明

- 主文件：`main.tex`
- 章节文件：`body/chapter1.tex` 至 `body/chapter5.tex`
- 摘要：`abstractC/abstractC.tex`、`abstractE/abstractE.tex`
- 附录：`appendix/appendix1.tex`、`appendix/appendix2.tex`
- 致谢：`acknowledge/acknowledge.tex`
- 图片：`pic/`

推荐编译方式：XeLaTeX。

```bash
xelatex main.tex
xelatex main.tex
```
